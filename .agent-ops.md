## ⚙️ Zorunlu Kurallar

1. **Her kod değişikliğinden sonra versiyonu güncelle** (`src/app/lib/siteConfig.js` → `version`)
   - Küçük UI/UX değişikliği → patch artır (1.7.6 → 1.7.7)
   - Yeni feature → minor artır (1.7.x → 1.8.0)
   - Breaking change → major artır (1.x → 2.0.0)
2. **Her değişikliği Changelog'a ekle** (bu dosyanın alt kısmı)
3. **Her işlemden sonra `get_errors()` çağır**, hata varsa düzelt
4. **Yeni component/lib dosyası oluştururken** `authorConfig`, `siteConfig`, `cityConfig` pattern'ını takip et (tek sorumluluk, export named)
5. **Tailwind** — `z-[N]` kullanma, `z-N` kullan; `min-w-[Npx]` yerine Tailwind arbitrary değeri veya scale
6. **Server/Client** — `"use client"` sadece gerçekten client state/event gerektiğinde
7. **Yeni API route** — her zaman `try/catch` + anlamlı HTTP status kodu döndür
8. **localStorage** — her zaman SSR-safe yaz (`typeof window !== "undefined"`)
